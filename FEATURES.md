# GymTracker — Feature Roadmap

## Szybkie ulepszenia
- [x] **Notatki do treningu** — szybka notatka po zakończeniu treningu (samopoczucie, kontuzje, uwagi)
- [x] **Zamiana ćwiczenia w trakcie treningu** — jeśli sprzęt jest zajęty, podmiana na alternatywne z tej samej partii mięśniowej
- [x] **Kopiowanie treningu** — powtórzenie ostatniego treningu jednym kliknięciem (bez planu)
- [x] **Superset / Giant set** — łączenie 2-3 ćwiczeń w jeden blok z jednym timerem przerwy
- [x] **Timer odliczający w trakcie serii** — do planków, izometryk itp.

## Analityka i progresja
- [x] **Personal records (PR)** — automatyczne wykrywanie i notyfikacja o nowych rekordach (max ciężar, max objętość, est. 1RM)
- [x] **Muscle heatmap** — widok tygodnia/miesiąca z kolorową mapą pokazującą które partie były trenowane i ile serii dostały
- [x] **Zmęczenie / readiness** — prosty formularz przed treningiem (sen, energia 1-5) + korelacja z wynikami
- [x] **Porównanie tydzień do tygodnia** — overlay wykresów z dwóch różnych tygodni tego samego planu

## Społeczność / motywacja
- [x] **Streak tracker** — seria treningowa z wizualizacją (jak GitHub contributions)
- [x] **Achievementy / odznaki** — np. "100 treningów", "Pierwszy PR", "5 treningów w tygodniu"
- [x] **Udostępnianie podsumowania** — generowanie ładnej grafiki z podsumowaniem treningu do stories/IG

## Bardziej zaawansowane
- [x] **Deload automatyczny** — po X tygodniach automatycznie sugeruj tydzień z obniżonym ciężarem
- [x] **Periodyzacja** — blokowa (siła → hipertrofia → deload) z automatycznym przełączaniem
- [x] **Body measurements** — tracking obwodów (ramię, klatka, udo, talia) + wykres zmian
- [ ] **Integracja z Apple Health / Google Fit** — sync wagi ciała i treningów *(wymaga natywnej apki)*
- [ ] **AI coach** — analiza historii i sugestie zmian w planie *(wymaga backendu z LLM)*

## Poprawki
- [x] **Kg/lbs per ćwiczenie** — zmiana jednostki dotyczy tylko danego ćwiczenia + automatyczne przeliczanie ciężarów
